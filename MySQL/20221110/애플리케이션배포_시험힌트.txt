- 애플리케이션배포 -
[필답형 시험]
1. 배포 파일
	jar(Java Archive), war(Web Archive), ear(Enterprise Archive)
2. Byte Code 언어
	Java, C#
3. 형상관리시스템 개념
	서비스 제공 대상 형상항목을 식별하여 기준선을 설정, 형상항목 변경 과정에서 점검, 검증등의 체계적인 통제를 통해 형 상항목 간의 일관성과 추적성을 확보하기 위한 시스템
4. 형상관리시스템 용어
	형상관리, 형상항목, 기준선, 마이그레이션, 리포지터리, 반출, 반입
5. 자바 테스트 프레임워크
	Junit 테스트 프레임워크
6. 소스코드를 점검하여 작성 규칙에 위반되는 소스코드를 추출/분석 도구
	소스코드 품질 측정도구(코드 인스펙션)
7. 버전관리 시스템 종류
	로컬 버전 관리, 중앙 집중식 버전 관리, 분산식 버전 관리
8. 버전관리 시스템 프로그램
	깃, 깃허브
9. 버전관리 시스템 용어
	init : 깃 저장소 생성
	add : stage 추가
	commit : 변경내용 저장
	status : 새로 추가된 파일목록 보기
	clone : 복제
